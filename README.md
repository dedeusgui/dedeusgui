## 📌 Projetos em destaque

### 🔹 Doitly — Rastreador de Hábitos com Gamificação

Aplicação web para acompanhamento de hábitos com **XP, níveis, conquistas e streaks**, construída com **PHP e MySQL**.  
O projeto utiliza **arquitetura em camadas, testes automatizados com PHPUnit e documentação técnica organizada**, e segue em evolução contínua como projeto pessoal.

### 🔹 The Social Network

Aplicação web full-stack simulando uma rede social, construída com **Node.js, Express e MongoDB**.  
O projeto explora **modelagem de API REST, autenticação com JWT, arquitetura modular (routes, controllers, services e repositories) e integração entre API e frontend estático**.
